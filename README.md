This repository contains the work completed during the VSD SoC Design and Planning workshop using the Sky130 PDK and OpenLANE flow.

Day 1: Introduction to the OpenLANE ASIC design flow and synthesis of the picorv32a design.

Day 2: Floorplanning concepts, utilization factor, aspect ratio, and placement of standard cells.

Day 3: Placement optimization, standard cell characterization, and analysis of design metrics.

Day 4: CMOS inverter layout design using Magic, SPICE extraction, LEF generation, and integration of a custom standard cell into the OpenLANE flow.

Day 5: Complete physical design flow including synthesis, floorplanning, placement, clock tree synthesis (CTS), routing, and layout visualization using Magic.
